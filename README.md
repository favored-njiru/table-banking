About The Project
KI-ARMA is a web-based group savings (chama) management system built for Kenyan communities. It allows a group treasurer to manage members, track contributions, record payments, and manage loans — all from a clean, modern dashboard.
The name KI-ARMA comes from the concept of communal saving and shared financial growth , a digital solution for the traditional chama system widely practised in Kenya.
Features
 Treasurer (Admin)

Secure login and registration
Full dashboard with group savings overview
Record, edit and delete member contributions
Manage group members — add, edit, view and remove
Track loan requests and repayments
View personal profile and contribution history
Change password securely
Role-based access control — only one treasurer per group

 Member

Secure login
Personal dashboard — view own contributions only
Cannot see other members' data
View own contribution history and status
View own profile
